PHP

Php pour Hypertext Preprocessor est un language de programmation (interpreted language),écrit en c datant des années 1994, utilisé principalement dans la programmation web, dernièrement plusieurs framewrok ont été développé à base de php (Laravel,Cakephp,Symfony,CodeIgniter ...)
Php peut communiquer avec plusieurs SGBD (MySQl , oracleDB, SQl Server ...) , plus de 79% des sites web à travers le monde ont été développés en php.

5.1 Affichage d'un message

pour afficher un message en php on peut utiliser les fonction et les constructeurs suivants:

Fonction / Constructeurs Utilisation
print print est une construction du language , elle prend une expression comme un argument.
echo echo est une construction du langage, elle prend une ou plusieurs experessions comme argument.
print_r permet d'afficher des informations lisibles concernant une variable données
var_dump permet d'afficher des informations concernant une variable données

5.2 Les variables en php

toute variable dans le langage php doit commencer par un $ , et elle ne doit pas contenr des caractères spéciaux (le underscore est autorisé _).
il n'est pas nécessaire de spécifier le type de variable à la déclaration , il sera déterminé à partir de la valeur initiale.

Exammple:

$test = "value"
$var2= 10
