JavaScript

    Il s’agit d’un langage de programmation de scripts basé sur les objets. Il sert à créer des pages Web interactives en manipulant le contenu des pages HTML. Il a été créé par Brendan Eich en 1995.
Grâce à la création du Node js en 2009 JavaScript peut être utilisé même en backend.

3.0 Introduction à la logique de Boole ?

dans le but de simplifier l'information aux maximum la notion de la logique de Boole a été introduit en 1854 par le mathématicien britannique George Boole. utilisé dans la premiére fois dans les circuits de commutation téléphonique .
Toute expression en logique ne peut avoir que l'une de ces valeurs { Vrai , faux } il existe plusieurs opérateurs pour l'évaluaion des expressions complexes parmi ces opérateurs on trouve ( && = et = * , || = ou = + )

3.2 Un Monde de Variable

imaginer que vous êtes un cuisinier et vous voulez préparer un plat pour votre restaurant, alors sûrement vous aurez besoin d'ingrédients, dans le cas de préparation d'un code les variables font le job !

Qu'est-ce qu'une variable ?

une est variable est tout simplement un espace mémoire réservé dans votre RAM par le code que vous avez executé ,
les variables sont identifiés par leur types qui peut être (un nombre entier , un nombre décimale , un caractére , boolean ,une chaine de caractéres ...)

3.3 Les opérations de bases

L'affectation :

l'affectation est l'opération la plus importante dans un langage de programmation elle sert à donner des valeurs a nos variable pour cela on utilise l'opérateur ' = '.

Les opérations de calcul :

Comme tout autre langage de programmation JavaScript propose les différentes opérations de calcul comme (l'addition(+), la soustraction (-) , la multiplication * , la division /)

3.4 Les conditions

Les conditions sert à faire des tests sur nos variable par exemple de savoir si une variable est égale à 0 ou bien supérieur ...
les conditions sont utilisés via le groupe d'instruction if else ;

3.5 Les boucles

Les boucles sert à répéter un groupe d'instruction plusieurs fois , en JS il existe trois maniére de les implémenter
while { ... groupes d'instructions }
for(conditions){ ... groupes d'instructions }
do{ } while();

3.6 Les tableaux

Un tableau est une structure de donnée qui permet de stocker plusieurs éléments.
