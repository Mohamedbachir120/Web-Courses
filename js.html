3.8 Programmation orientée objets

Nous avons vu jusqu'à présent que des simples variables (entier , Chaine de caractères , boolean , ...) avez vous pensé comment modéliser un personnage d'un jeu , un employé d'une entreprise , une voiture de course ...
Eh oui bien sûr que c'est possible via ce qu'on appelle la programmation orientée objet, ce principe est basé sur la création des models (classes) qui possède des caractéristiques (attributs) et des fonctions qui sert à réaliser des opérations précises.

class User
{

    name!: String;
    last_name!: String;
    age!: Number;

constructor (name: String ,last_name: String ,age: Number )
  {
    this .name = name;
    this .last_name = last_name;
    this .age = age;
  }
sePresenter()
  {
    alert ("je m'appelle"+this.name!+" "+this.last_name+ "j'ai"+this.age);
  }
}
const user: User = new User ("mohamed","bachir",23);
user.sePresenter();
