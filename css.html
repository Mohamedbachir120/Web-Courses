CSS3

Les feuilles de style en cascade généralement appelées CSS est un langage informatique qui sert à décrire la présentation et le style d’un document HTML et XML.
Datant des années 1990.

2.1 Comment appliquer un style ?

Le code CSS peut être introduit avec plusieurs méthodes, une de ces méthodes est les feuilles de styles internes on utilise la balise <style> … </style>.


2.2 Les sélecteurs CSS 

Les sélecteurs sont la première partie d'une règle CSS. Il s'agit d'une syntaxe servant à identifier les éléments du document auxquels la règle s'appliquera.

Le type de selecteur Example
balise p , div , ul , h1
classe .maclasse
id #monid
universel *
Parent enfant ul li, tr td


2.3 Les propriétés CSS 

  • Les propriétés sont des paires de clés et valeurs.
  • La clé est le nom de la propriété, suivie des deux points, terminée par la valeur qui décrit la propriété.
Propriété Valeurs possibles
Font-family Nom de la police
font-size Taille du texte (15 px , 10% ….)
Font-weight Bold,normal,100 … 800
Text-decoration Overline,line-through,underline
Text-align Center , right , left , justify
Color rgb(0-255,0-255,0-255) , Red,blue,green,yellow ...
Background-color  Rgb(20,34,57), linear-gradient(120deg,blue,red)
border solid 1px black
Box-shadow 0 0 3px 3px red
Border-radius 10px,16%
background-image url("la source")
height 100px,100%
width 100px,100%
padding , margin 10px

2.4 Les flex Box

Le principe de la mise en page avec Flexbox est simple : vous définissez un conteneur, et à l'intérieur vous placez plusieurs éléments. Imaginez un carton dans lequel vous rangez plusieurs objets : c'est le principe ! Pour utiliser la propriété du flexbox on utilise la règle  :
display :flex ;
pour spécifier l’ordre (horizontal , vertical ) :
flex-direction : row ou bien column .
